6. Images libres

Oui, on peut ajouter une interface d’upload pour préparer le visuel. En revanche, pour un affichage fiable chez les destinataires, les images doivent idéalement être hébergées sur une URL publique HTTPS. Les images encodées directement dans le HTML du mail peuvent mal passer dans Gmail ou certains clients mail.

7. Actions

Le bouton Copier le message essaie de copier directement la version riche pour collage dans Gmail. Selon Chrome et le contexte local, ça peut être bloqué. Dans ce cas, utilise l’aperçu seul puis copie-colle visuellement, ou le HTML.
